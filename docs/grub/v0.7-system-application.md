# GRUB v0.7 — System Application

## Objetivo

Promover o tema NeoVolt v0.5 do repositório para o sistema real.

---

## Local do projeto

themes/grub/

---

## Local do sistema

/boot/grub/themes/NeoVolt/

---

## Backup

Antes de qualquer alteração:

```bash
sudo cp /etc/default/grub /etc/default/grub.bak
sudo cp /boot/grub/grub.cfg /boot/grub/grub.cfg.bak
sudo cp /boot/grub/themes/NeoVolt/theme.txt \
/boot/grub/themes/NeoVolt/theme-v0.4.txt
```

---

## Aplicação

Copiar:

```bash
sudo cp themes/grub/theme.txt \
/boot/grub/themes/NeoVolt/theme.txt
```

---

## Atualização

```bash
sudo update-grub
```

---

## Validação

Verificar:

- Linux Mint
- Windows Boot Manager
- UEFI Firmware Settings

---

## Rollback

Restaurar:

```bash
sudo cp /boot/grub/themes/NeoVolt/theme-v0.4.txt \
/boot/grub/themes/NeoVolt/theme.txt

sudo update-grub
```
