# NeoVolt Boot Icons v1.1

## Objective

Create a unified icon set for the NeoVolt Boot Manager.

## Design Principles

* Transparent background
* NeoVolt green accents
* High contrast
* Readable at GRUB resolution
* Consistent style across all icons
