# NeoVolt Icon Identity v1.1

## Purpose

Define the official visual identity for all icons used in the NeoVolt Linux Experience ecosystem.

## Official Design Decision

All NeoVolt Boot Manager icons shall be derived from the official NeoVolt brand symbol:

```text
N ⚡ V
```

This symbol becomes the visual foundation for:

* GRUB Boot Manager
* Login Screen
* Plymouth Splash Screen
* NeoVolt Control Center
* NeoVolt Workspace
* Future NeoVolt Applications

## Design Principles

* Consistent visual language
* NeoVolt green as primary accent color
* White as secondary color
* Dark backgrounds preferred
* High contrast
* Readable at small resolutions
* Optimized for GRUB environments
