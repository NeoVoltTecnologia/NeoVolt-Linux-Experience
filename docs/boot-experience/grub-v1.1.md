# NeoVolt Boot Experience 1.1

## Objetivo

Evoluir a experiência de inicialização do notebook NeoVolt para uma identidade visual contínua, profissional e alinhada à NeoVolt Tecnologia.

A meta é reduzir quebras visuais entre GRUB, mensagens do kernel, LUKS, Plymouth e login.
