# NeoVolt Technical Toolkit v1.0

## Objective

Create a professional field toolkit for NeoVolt technicians focused on security systems, networking, remote access and technical diagnostics.

## Philosophy

The NeoVolt Technical Toolkit is not a separate operating system.

It is a curated collection of tools integrated into the NeoVolt Linux Experience.

## Network Diagnostics

### Purpose

Discover, analyze and troubleshoot networks.

### Planned Tools

* Nmap
* Angry IP Scanner
* Wireshark
* Netcat
* Traceroute
* DNS Utilities
* Speedtest CLI

## CCTV and IP Cameras

### Purpose

Support installation and maintenance of security systems.

### Planned Tools

* ONVIF Discovery
* RTSP Stream Tester
* VLC
* FFmpeg
* Camera Identification Tools

## Remote Access

### Purpose

Access client systems remotely.

### Planned Tools

## Storage and Recovery
