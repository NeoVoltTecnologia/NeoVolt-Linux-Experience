# NeoVolt Network Discovery v0.2

## Objective

Extend the Network Discovery tool to identify nearby devices on the local network.

## New Features

### ARP Discovery

Display devices known by the local ARP table.

Commands:

```text
arp -a
ip neigh
```

### Device Identification

Show:

* IP address
* MAC address
* Interface

## Expected Benefits

* Find IP cameras
* Find DVRs and NVRs
* Find routers
* Find access control devices
* Accelerate field diagnostics

NeoVolt Technical Toolkit v0.2
